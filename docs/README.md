# SGHSS - Sistema de Gestão Hospitalar e de Serviços de Saúde

Projeto desenvolvido para a disciplina Projeto: Desenvolvimento Back-end

## Tecnologias

- Node.js
- Express
- JavaScript

## Funcionalidades

- Cadastro de pacientes
- Cadastro de médicos
- Agendamento de consultas
- Prontuários
- Receitas
- Controle de leitos
- Login
- Relatórios

## Estrutura

/controllers

/routes

/models

/middleware

/services

/utils

## API

GET /pacientes

POST /pacientes

PUT /pacientes/:id

DELETE /pacientes/:id

Mesmo padrão para:

- médicos
- consultas
- prontuários
- receitas
- leitos
